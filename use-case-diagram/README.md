# Use Case: User Registration

**Actor:** User  
**Goal:** Register a new account in the system  

**Preconditions:**  
- User has a valid email address  
- User can access the registration page  

**Main Flow:**  
1. User navigates to the registration page  
2. User fills in required fields (name, email, password)  
3. User submits the form  
4. System validates input and creates an account  
5. System sends a confirmation email  

**Alternative Flow:**  
- If email already exists, system shows an error  

**Postconditions:**  
- User account is created and active
