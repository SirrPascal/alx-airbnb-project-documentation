# Use Cases

## 1. User Management
- **As a new user**, I want to register an account (as a Guest or Host) so that I can access all platform features (booking, listing, etc.).
- **As a registered user**, I want to be able to log in using email/password so that I can access my personalized dashboard.
- **As a user**, I want to be able to update my profile details and photos so that I can present accurate information to the community.
- **As a user**, I want to be able to securely log out so that I can end my session and protect my data.

## 2. Property Management (Host)
- **As a Host**, I want to be able to create a new property listing with photos, details, and pricing so that guests can find and book my property.
- **As a Host**, I want to be able to modify my existing listing details, availability, or pricing so that my listing always reflects the current offering.
- **As a Host**, I want to be able to remove a property listing so that it no longer appears in search results.
- **As a Host**, I want to be able to view a list of upcoming and past bookings for my properties so that I can manage my calendar and guest arrivals.

## 3. Search and Booking (Guest)
- **As a Guest**, I want to be able to search properties using multiple filters (location, price, dates, amenities) so that I can quickly find a property that meets my needs.
- **As a Guest**, I want to be able to view full property details, photos, and reviews so that I can make an informed decision before booking.
- **As a Guest**, I want to be able to select dates and confirm a booking for a property so that I can reserve my stay.
- **As a Guest or Host**, I want to be able to cancel a booking based on the policy so that appropriate refunds and availability updates are processed.
- **As a Guest**, I want to be able to submit a review and rating for a completed stay so that I can share my feedback and help other guests.

## 4. Payment Management
- **As a Guest**, I want to be able to pay securely via the payment gateway so that my booking is confirmed and my financial data is protected.
- **As a Host**, I want to automatically receive my payment after a guest's stay is completed so that I can get compensated for my service.
- **As the System**, I want to be able to reverse payments when a valid cancellation occurs so that the Guest receives the correct refund.
